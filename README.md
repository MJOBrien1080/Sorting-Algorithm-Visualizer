# Sorting-Algorithm-Visualizer
Python-based sorting algorithm visualizer that demonstrates how different sorting algorithms work step-by-step. It generates a list of random numbers and visually animates the sorting process using bar charts and colors. Built for visualization understanding how sorting alorithms compare visually by precticing with Matplotlib animations.
Using Python 3 with Matplotlib (for visualization)and Random module (for generating data).
Install Python verions 3 or higher, run python terminal and run "pip install matplotlib" they it is able to be ran.
